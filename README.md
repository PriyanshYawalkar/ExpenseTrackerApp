# Expense Tracker

Expense Tracker is a fully functional iOS app built entirely with SwiftUI. It allows users to track their expenses, manage budgets, and gain insights into their spending habits. ##### Data Persistence: Utilizes Core Data for reliable data storage.
##### User-Friendly Interface: Designed with simplicity and ease of use in mind.
